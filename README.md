# Mercedes-Benz Sales Analysis (2020-2025)
## Data Analytics & Economic Insights Project

### Project Overview
This project analyzes a dataset of over 12 million rows of Mercedes-Benz sales data. As an economist and data enthusiast, I've explored the intersection of market dynamics and vehicle performance using Python.

### Key Features
* **Big Data Processing:** Efficiently handled 12M+ rows using **Parquet** format and **NumPy** vectorization.
* **Economic Metrics:** Engineered a custom **"Price per HP"** index to measure technological efficiency.
* **Visualizations:** 8 high-resolution charts created with **Seaborn** and **Matplotlib** highlighting market trends, fuel distribution, and price dynamics.

### Technical Stack
- **Python** (Pandas, NumPy)
- **Data Visualization** (Matplotlib, Seaborn)
- **Storage:** Apache Parquet

### Top Insights
1. **Strategic Pricing:** Observed a clear shift toward high-margin luxury models and EV/Hybrid integration.
2. **Performance Correlation:** Strong positive correlation between Horsepower and Base Price, with specialized trends for Turbo-charged engines.
3. **Market Preferences:** Black and White remain the dominant color choices, while regional sales show distinct preferences for Turbo models.
