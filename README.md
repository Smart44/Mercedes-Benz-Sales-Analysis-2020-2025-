# Mercedes-Benz-Sales-Analysis-2020-2025-
Automated data analysis and visualization of Mercedes-Benz sales (2020-2025) using Python, Pandas, and Matplotlib.

## Overview
This project provides an automated pipeline to process large-scale automotive sales data. It cleans raw CSV data, performs statistical aggregation, and generates visual insights within seconds.

## Key Features
* **High Performance:** Processes years of sales data in under 25 seconds.
* **Automated Visualization:** Generates distribution charts by Model and Fuel Type.
* **Structured Output:** Automatically organizes results into a dedicated output folder.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, OS
